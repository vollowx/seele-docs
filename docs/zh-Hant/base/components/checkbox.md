# Checkbox（複選框）

- 混合：[`InternalsAttached`](../mixins/internals-attached.md)
- 混合：[`FormAssociated`](../mixins/form-associated.md)

## 屬性

| 名稱      | 類型    | 默認值  | 描述                   |
| --------- | ------- | ------- | ---------------------- |
| `checked` | Boolean | `false` | 複選框是否被選中。     |

## 事件

| 名稱     | 描述                     |
| -------- | ------------------------ |
| `change` | 當複選框狀態改變時觸發。 |
