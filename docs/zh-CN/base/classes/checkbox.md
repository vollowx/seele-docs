# Checkbox

- 混入：[`InternalsAttached`](../mixins/internals-attached.md)
- 混入：[`FormAssociated`](../mixins/form-associated.md)

## 属性

| 名称      | 类型    | 默认值  | 描述               |
| --------- | ------- | ------- | ------------------ |
| `checked` | Boolean | `false` | 复选框是否被选中。 |

## 事件

| 名称     | 描述                     |
| -------- | ------------------------ |
| `change` | 当复选框状态改变时触发。 |
